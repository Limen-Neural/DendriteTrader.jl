## Issue #25 — Fix win_rate calculation

**File:** `src/backtest/Backtest.jl`, lines 183-184

**Current (buggy):**
```julia
closed_trades = filter(t -> t.pnl != 0.0, trade_log)
wr = isempty(trade_log) ? 0.0 : length(closed_trades) / length(trade_log)
```

**Problems:**
1. Denominator is `trade_log` (all trades), not `closed_trades` — dilutes metric
2. Numerator counts all closed trades, not just profitable ones

**Fix:**
```julia
closed_trades = filter(t -> t.pnl != 0.0, trade_log)
winning_trades = filter(t -> t.pnl > 0.0, closed_trades)
wr = isempty(closed_trades) ? 0.0 : length(winning_trades) / length(closed_trades)
```
